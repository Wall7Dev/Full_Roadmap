Combining CSS Selectors

There are several ways of combing CSS selectors for styling our html document

  • Way 1: Grouping

    An example is;

                h1, .className{ css styling goes herer} 
    
            h1, h2 { css styling goes here}

  • Way 2: Child

    An example is;

                  h1 > className{ css styling goes herer} 
    
              h1 > h2 { css styling goes here}

  • Way 3: Desendant Selector

    An example is;

                  selector Selector{ css styling goes herer} 
    
              ancestor desendent { 
                css styling goes here
              }

  • Way 4: Combining combiners

    An example is;

                  selector Selector{ css styling goes herer} 
    
              ancestor desendent { 
                css styling goes here
              }

  • Way 5: Chaining Selectors

    An example is;

                  selectorSelector{ css styling goes herer} 
    
              h1.box#idName { 
                css styling goes here
              }
