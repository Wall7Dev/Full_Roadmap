CSS Concepts

There are 5 ways to select tags in CSS, these are;

-Tag selelctors
example: h2{
color:red;
background-color:black;
}

-Class selectors
.className{
color:red;
background-color:black;
}

-ID selectors
#uniqueName{
color:red;
background-color:black;
}

-Universal Selectors
\*{
color:red;
background-color:black;
}

-Attribute Selector
p[href]{
color:red;
background-color:black;
}
